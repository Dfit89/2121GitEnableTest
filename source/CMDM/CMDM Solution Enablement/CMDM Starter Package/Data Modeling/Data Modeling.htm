Data Modeling

With the help of this guide, users should be able to model the following common customer data challenges in STEP:

  • Individual and Organization Customers
  • Individuals as Contact Persons
  • Contact Persons in relation to Organization Customers
  • Addresses and Contact Information specific to a Customers and Contact Persons

Additional data may very well be applicable to include in a CMDM solution, but is not covered by these guidelines.

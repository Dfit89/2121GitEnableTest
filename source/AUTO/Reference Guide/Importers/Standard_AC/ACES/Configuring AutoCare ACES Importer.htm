Prerequisites

It is expected that anyone configuring the AutoCare ACES Import solution within a Web UI be familiar with the Web UI Designer, as basic concepts for working with the designer are not covered in this section. In addition, the user must have appropriate privileges to access the designer. For more information, see the Designer Access topic (here) within the Web UI Getting Started section of STEP Online Help.

Anyone configuring the AutoCare ACES Importer is expected to be familiar with the Importing Automotive Data section here, as basic concepts for working with an automotive importer are not covered in this section.

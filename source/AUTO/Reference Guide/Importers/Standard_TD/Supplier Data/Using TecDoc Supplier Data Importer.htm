Using TecDoc Supplier Data Importer

A TecDoc Supplier Data file can be imported into STEP by uploading it to either a configured hotfolder on an application server, or through a File Loading Widget on a Web UI Homepage.

The following information is available for the TecDoc Supplier Data Importer.

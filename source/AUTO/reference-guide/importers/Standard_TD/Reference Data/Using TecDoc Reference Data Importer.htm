Using TecDoc Reference Data Importer

A TecDoc Reference Data file can be imported into STEP by uploading it to either a configured hotfolder on an application server, or through a File Loading Widget on a Web UI Homepage.

The following topics provide information on using the TecDoc Reference Data Importer.

  • Error Handling for Reference Data Imports (here)
